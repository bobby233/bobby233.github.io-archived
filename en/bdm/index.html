BDM(Bobby Dark Mode)

Author：Bobby233

Translator：Bobby233

INTRO

Did you see the UI of this website? Isn't that cool? This is the HTML which is written with BDM(Bobby Dark Mode). BDM is very easy to use. Simply copy the code of style or link the CSS file to use it. The latest version is v0.0.0 Original Edition. Welcome to give suggestions to us by e-mailling the developers.

Since the special license of BDM, only the website which has written "UI by BDM"(there's a HYPERLINK on BDM) or "UI powered by BDM" can use BDM open-source. If you see a website which hasn't written these and uses the BDM, report this to the developers IMMEDIATELY.

COLORS

The BDM's core is the colors. Me, the inventor of BDM, tried a large amount of times and finally get these colors below. I recommand you to use these colors in the Dark Mode of your personal blog. The formatting below is: VISUAL_COLOR HEX/RGB/NAME ELEMENT_TO_USE

  • #fff h1/h2/h3...

  • rgb(163,163,163) p/li

    Recommended!

  • #03e9f4 a/button

    Recommended!

  • #191919 body{background-color}

    Recommended!

  • gray footer a

    Recommended!

These are some representive examples of BDM. We will write every kinds of colors later.

SPECIAL EFFECTS

In a great UI, only colors is not enough. Did you see the sparkling button on the top? It uses the buttons' shadow of BDM. These special effects will be discussed below.

  • The unchoosed buttons: #03e9f4 is the background, #191919 is the text, no border and choosed border. Code: 
    button {
    background-color: #03e9f4;
    color: #191919;
    border: none;
    outline: none;
}
