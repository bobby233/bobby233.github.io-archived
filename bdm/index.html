BDM(Bobby Dark Mode)

作者：Bobby233

简介

看到这个网站的UI了吗？很炫酷吧。这就是使用BDM(Bobby Dark Mode)写出的HTML，BDM十分易用，只需要复制style的代码或者远程调用CSS即可立即开始使用。最新版本为v0.0.0初始版，如有不足请向开发者提出建议。

由于BDM的独特开源许可，只有在网页脚注（footer）部分标明UI by BDM（BDM有超链接）或UI powered by BDM的才可以开源使用BDM，如果看到没有标注却使用类似UI的网站，请立即向开发者报告。

配色

BDM的核心肯定就是配色了，以下这些配色是创始人Bobby233经过反复测试后得出的，十分推荐使用于个人博客的Dark Mode。以下配色格式如：颜色视觉展示 颜色十六进制/rgb/名称 使用元素

  • #fff h1/h2/h3...

  • rgb(163,163,163) p/li

    Recommended!

  • #03e9f4 a/button

    Recommended!

  • #191919 body{background-color}

    Recommended!

  • gray footer a

    Recommended!

以上只是列举了一些比较标志性的配色，所有的配色将在之后完善。

特殊效果

在一个优良的UI中，只有配色是完全不够的，看到顶部的那个发光的按钮了吗？那使用了BDM的按钮阴影，这些特殊效果将会在下面详细描述。

  • 未选择的按钮：本身使用#03e9f4为背景，#191919为文字颜色，没有边框和选中后边框。代码如下： 
    button {
    background-color: #03e9f4;
    color: #191919;
    border: none;
    outline: none;
}
  • 其他正在补充中……

  • BDM本身开源，但是需要提前向Bobby233发邮件声明，绝大多数情况都会同意。如果没有声明并且使用属于侵权，需要立即删除技术。
  • 完整使用技术时必须严格遵守邮件内容，需要在footer添加UI by BDM。
  • BDM允许基于此技术添加代码，在这样做时，需要在footer添加UI powered by BDM。
  • BDM有权无理由收回不当网站的BDM技术。

  • BDM有权使用基于BDM开发的技术来改进BDM本体。
